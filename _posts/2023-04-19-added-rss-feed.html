Added an RSS feed

Now that Mastodon is becoming more of a thing, I'm starting to use RSS feeds to post to my account there.

The other day, I used MastoFeed to post my Glass photos via RSS and it worked really nicely.

This inspired me to set up an RSS feed for my blog here (even though I don't write much). Because I'm using Jekyll, it was just a super easy config change and all my posts were set up in a feed.

Hopefully when I commit and push this post to GitHub Pages it automatically gets posted to Mastodon. If you're reading this post, you can go check my account to see if it worked or not.
